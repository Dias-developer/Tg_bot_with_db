# Телеграм бот с Базой данных 'учет клиентов'
Простой Telegram-бот, разработанный на Python с использованием Aiogram 3 и SQLite3. Проект демонстрирует реализацию CRUD-операций (создание, чтение, обновление и удаление данных), а также работу с загрузкой файлов.

🚀 Возможности
CRUD-операции с базой данных SQLite
Работа с Telegram Bot API через Aiogram 3
Простая и масштабируемая архитектура проекта
Возможность дальнейшего расширения функционала

🛠️ Используемые технологии
Python 3.13+
Aiogram 3
SQLite3
asyncio

⚙️ Установка
1. Клонируйте репозиторий
git clone https://github.com/Dias-developer/Tg_bot_with_db.git

2. Перейдите в папку проекта
cd Tg_bot_with_db

3. Создайте виртуальное окружение
python -m venv .venv

4. Активируйте виртуальное окружение

Windows
.venv\Scripts\activate

Linux / macOS
source .venv/bin/activate

5. Установите зависимости
pip install -r requirements.txt

🔧 Настройка

Создайте файл .env в корневой директории проекта:
BOT_TOKEN='ВАШ_ТОКЕН_БОТА'

▶️ Запуск
python main.py
